Was ist TranDumper?
TranDumper ist einFREEWARE-Parser der Braindumps und andere Multiple-Choice Dateien als Prüfung darstellt.
This program konvertiert Braindumps (und andere .txt-Dateien) in offline Prüfungssimulationen. Any modification or derived works made from TranDumper (code or dump text file) must be released as freeware.

You can help to localize the user interface of TranDumper. Localized files (TD_Readme.html, Trandumper.html) must be located in a country 2-letters code folder.
For now, Trandumper is ready to receive: DE, FR, SP, IT.

The 3.0x version is released along with a German version translated by Bouchi. Please inspire from his template to make your own localized version.

Thank you for using TranDumper.


How to format text file for TranDumper?
For examples of how to use TranDumper, you can see the sample directory. You can read braindumps with little or no modifications to the dumps. See sample_plain.txt for a plain dump file, or see sample_enhanced.txt that explains the use of enhanced features.

You should only have to make minor formatting changes to your dumps to be sure that TranDumper will read them properly. You simply need to save the dump web page as plain text (*.txt) and then check your dump file format as follow:
  1. Subject must begin with a number and a dot on a new line.
    EXAMPLES: 1.  or  99. (Maximum number of questions: 400)

  2. A blank line is mandatory between the question text and the multiple-choice answers

  3. Select options must begin with English Alpha and a dot or a ")".
    EXAMPLES: a.  a)  A. (Maximum number of answers: 15)

  4. ANSWER MUST BE ON ONE LINE.
    The Answer format must be written as follows to be properly recognized.
    EXAMPLES: Ans:a,c   Answer: b    Ans : bc   Answer: b,c and d

  5. If there is any comment or explanation for the answer, a blank line is mandatory between the "Answer:" line and the explanation.

  6. All the Selection options and answer are case-insensitive. That is, both Ans: and ANS: are legal syntax.


Minimum Requirements
Windows 95, 98, NT, 2000, Internet Explorer 5.0 or greater and screen resolution of 800x600.
TranDumper can only be run on IE compatible browser. It uses an ActiveX (File System object of scripting engine to open local file). And also because Trandumper.html contains many VBscripts.
  1. Be sure that you have Internet Explorer 5.0 or greater installed. TranDumper CANNOT run on Internet Explorer 3.x OR 4.x.
  2. Because it must open local files, it's recomended that the security settings in your browser should be set to medium or below.
  3. It can occur that Internet Explorer ask to confirm the trust of the ActiveX that is running. You should confirm by clicking "Yes" to proceed
  4. Don't use PWS or IIS to run this program!
  5. The install diretory shouldn't be any "virtual" PWS or IIS directory. For example use the default instalation directory!
